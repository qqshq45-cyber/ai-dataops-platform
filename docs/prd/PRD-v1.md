# PRD v1

## Product Overview
AI DataOps Platform for evaluating AI-generated videos in enterprise workflows.

## Goals
- Provide a structured workflow for video generation evaluation.
- Support configurable rule-based review and scoring.
- Enable collaboration between reviewers, operators, and admins.

## User Roles
- Admin
- Operator
- Reviewer

## Core Features
- Project management
- Video submission and review
- Rule-based evaluation engine
- Metrics and reporting
- Audit trail and status tracking

## Non-Goals
- Fully automated video generation
- Generic social media content management

## Success Metrics
- Review turnaround time reduced
- Evaluation consistency improved
- Platform adoption across teams
