# AI DataOps Platform - Development Guide

## Project Goal

Build an enterprise AI DataOps Platform for AI video generation evaluation.

This project is based on a real business workflow instead of a demo.

---

## Tech Stack

Backend:
- Python
- FastAPI
- SQLAlchemy
- Alembic
- MySQL

Frontend:
- React
- TypeScript
- Vite

AI:
- LangGraph
- OpenAI Compatible API

Deployment:
- Docker
- Docker Compose

---

## Principles

1. Explain before coding.
2. Keep modules independent.
3. Avoid hard-coded business rules.
4. Build a configurable Rule Engine.
5. Follow Clean Architecture.

---

## Coding Style

- Small commits
- Clear naming
- SOLID principles
- RESTful APIs
- Type hints for Python
